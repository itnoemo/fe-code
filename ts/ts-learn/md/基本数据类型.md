# 布尔型

关键 boolean

```typescript
let b: boolean = false
console.log(b)
b = true
console.log(b)

// false
// true
```



# 数字

关键字 number

```typescript
let num: number = 1
console.log(num)
// 1
```



# 字符串类型

关键字 string

```typescript
let str: string = 'str'
console.log(str)
let str1: string = "str1"
console.log(str1)

// str
// str1
```
